Phased Array Ultrasonics

Phased array ultrasonic testing is based on principles of wave physics, which also have applications in fields such as optics and electromagnetic antennae. The beam from a phased array probe can be focused and swept electronically without moving the probe, because it is made up of multiple small elements, each of which can be pulsed individually at a computer-calculated timing.
